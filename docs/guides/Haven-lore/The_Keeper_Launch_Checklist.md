## TROUBLESHOOTING

### Problem: No discoveries reported
**Solution:**
- Post direct challenge: "Find me ancient bones on [Region]"
- Post Keeper query: "I sense something on [Type] planets, can you verify?"
- Create event: "Discovery Bounty Week"

### Problem: Same discovery type repeatedly (no variety)
**Solution:**
- This is fine! It's a pattern waiting to be recognized
- Create mini-investigation around it
- Challenge players: "Try to find [different type]"

### Problem: Contradictory discoveries
**Solution:**
- This is a feature, not a bug
- Keeper: "These discoveries contradict. Which is true? Both?"
- Creates intrigue and deeper investigation

### Problem: Players losing interest
**Solution:**
- Post mysterious fragment to spark curiosity
- Create urgent-feeling challenge
- Reveal partial answer to long mystery
- Post humorous Keeper reflection

### Problem: Too many discoveries to process
**Solution:**
- That's success! Ask for mod help
- Batch-process responses (post 3-4 at once)
- Create player researcher groups to help analyze
