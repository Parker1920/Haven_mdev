**When player finds:** Ancient bones at excavation site

**Keeper Response Template:**
```
[KEEPER//PALEONTOLOGICAL_DISCOVERY]

> [Structure suggests predator/prey/unusual adaptation]

> [Connection to other discoveries?]

**Pattern Opportunity:** Same species on multiple planets = seeding/transport

**When player finds:** Log at facility/terminal

**Keeper Response Template:**
```

## SECTION 5: BUILDING MYSTERY TIERS

### TIER 1: Initial Discovery (Week 1-2)
```
- Individual discoveries arrive
- Keeper acknowledges and analyzes each
- Archive grows
- Players feel their findings matter
```

### TIER 2: Pattern Recognition (Week 2-3)
```
- Multiple similar discoveries noted
- Keeper announces pattern
- Community awareness increases
- New "investigation thread" opens
```

### TIER 3: Hypothesis Formation (Week 3-4)
```
- Keeper proposes theories
- Multiple interpretations possible
- Community debates
- Players explore to gather more evidence
```

### TIER 4: Deeper Mystery (Week 4+)
```
- As players find more, larger mystery emerges
- Initial theories confirmed/challenged/modified
- New questions arise from answers
- Story branches into multiple directions
- Long-term investigation threads (months+)
```
